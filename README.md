## About the app

**The Best Wine Choice** helps you discover, learn about, and choose wines with confidence. The app uses AI and smart features to personalize your experience and surface recommendations that match your taste.

### Features

- **Label scan** — Point your camera at a wine label to get details, tasting notes, and pairing ideas.
- **Personalized recommendations** — Recommendations improve as you scan and rate wines.
- **Favorites** — Save wines to a collection and access them from your profile.
- **Nearby places** — Find wine shops and restaurants near you (with location enabled).
- **Tasting notes & pairings** — Explore taste profiles and get food-pairing suggestions.

### Who it’s for

The app is for anyone who wants to explore wine without guesswork: casual drinkers, enthusiasts, and people who want quick, reliable info when shopping or dining.

## Deploying to GitHub Pages

1. **Clone and open the repo** (if needed):
   ```bash
   git clone https://github.com/tundesalisu/mybestwinechoice.git
   cd mybestwinechoice
   ```

2. **Commit and push** after any edits:
   ```bash
   git add .
   git commit -m "Your update message"
   git push origin main
   ```

3. **Turn on GitHub Pages** (one-time):
   - Repo → **Settings** → **Pages**
   - **Source**: Deploy from branch
   - **Branch**: `main` (or the branch you use)
   - Save. The site will be at `https://tundesalisu.github.io/mybestwinechoice/`

Updates pushed to the selected branch go live within a few minutes.
